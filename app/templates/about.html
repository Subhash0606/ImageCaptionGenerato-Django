Image captioning is the process of generating caption i.e. description from input image. It requires both Natural language processing as well as computer vision to generate the caption. The popular benchmarking dataset which has images and its caption are:

  • Common Objects in Context (COCO). A collection of more than 120 thousand images with descriptions.
  • Flickr 8K: A collection of 8 thousand described images taken from flickr.com.
  • Flickr 30K: A collection of 30 thousand described images taken from flickr.com.
